BrightHealth Medical Group

BrightHealth is a brand concept for a medical group based in San Diego. For this project I created a brand identity and icon set using the golden ratio. I implemented a cross in the logo to distinguish it as a medical company. The colors represent protection, help, and trust. The icon set would be used for supporting brand elements and wayfinding signage.
