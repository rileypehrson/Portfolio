Fall Bedding Set

This fall bedding set was an illustration project I created by turning a set of icons into a seamless pattern. I curated a muted fall color palette for an audience of adults ages 18-40. I took inspiration from folk art to illustrate the hero owl and added supporting illustrations to complement the aesthetic. This pattern would be used for blankets, pillows, and bedding.
